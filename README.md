# 二狗吉他谱

## 简介

简单易用无广告  
方便快捷响应式  

查看地址：[http://guitar.hoojoe.com/](http://guitar.hoojoe.com/)

欢迎踊跃投稿!

## 添加吉他谱

 1. 创建 `歌手文件夹` 新增 `歌名.md` 文件
 2. 在 `歌名.md` 中添加吉他谱图片（直接用其它站的图片链接或任何可访问的链接）
 3. 在 `_sidebar.md` 添加该吉他谱的链接
 4. 推送
